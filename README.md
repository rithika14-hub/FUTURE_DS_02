# Customer Churn Analysis Dashboard

## Project Overview
This project focuses on analyzing telecom customer churn data to understand customer behavior, retention patterns, and factors influencing churn using Python and Microsoft Excel.

## Objectives
- Analyze customer churn behavior
- Identify key retention drivers
- Understand customer segments and trends
- Create interactive dashboards
- Generate business recommendations

## Tools & Technologies
- Python
- Google Colab
- Microsoft Excel
- Pivot Tables & Charts

## Project Workflow
1. Data Cleaning & Preprocessing
2. Exploratory Data Analysis (EDA)
3. Churn Trend Analysis
4. Dashboard Creation
5. Insight Generation

## Dashboard Features
- Churn Distribution Analysis
- Contract Type Analysis
- Internet Service Analysis
- Payment Method Analysis
- Customer Tenure Analysis
- Interactive Slicers

## Key Insights
- Month-to-month contracts showed higher churn
- New customers had high churn rates
- Fiber optic users showed increased churn
- Electronic check users were more likely to churn
- Long-term customers had better retention
